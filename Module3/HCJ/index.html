HTML

HTML stands for Hyper Text Markup Language. Hypertext means that the document contains links that allow the reader to jump to other places in the document or to another document altogether. The latest version is known as HTML5. A Markup Language is a way that computers speak to each other to control how text is processed and presented. To do this HTML uses two things: tags and attributes. Tags are used to mark up the start of an HTML element and they are usually enclosed in angle brackets. An example of a tag is: <h1>. Many HTML tags have semantic meaning. That is, the element itself conveys some information about the type of content contained between the opening and closing tags.

CSS

CSS Rules: Apply same style rules on multiple elements. Can control the presentation of multiple pages of a website with a single style sheet. Used to style dynamic states of elements such as hover, focus, etc. that isn't possible otherwise. Used to change the position of an element on a web page without changing the markup. Used to alter the display of existing HTML elements. Used to transform elements like scale, rotate, skew, etc. in 2D or 3D space. Used to create animations and transitions effects without using any JavaScript. Used to create print friendly version of your web pages.CSS uses <link> elements to apply external stylesheets and <style> elements to apply internal stylesheets to HTML.

JAVASCRIPT

JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images. JavaScript is still considered an interpreted language, since the compilation is handled at run time, rather than ahead of time.
